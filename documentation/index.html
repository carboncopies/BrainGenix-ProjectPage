BrainGenix Documentation

Note

As BrainGenix is still in development, we wanted to make sure that you were aware that it does not have all the features outlined implemented yet. You are welcome to contribute if you're able to help us get closer to finishing version one. If you'd like to see a more detailed view of the current status, we invite you to check out our Tech Spec and Docs page below.



Tech Spec

Our Technical Specifications document is available here via google docs. It contains a more in-depth outline of everything going on in the project and what the features will be. As with all other new projects, the feature sets and data contained within this document is subject to change.



Working Documentation

Our current working documentation is available here via google docs. This document is not guarenteed to be up-to-date with the current codebase. As we write our code and implement features needing documentation, this page is updated along with it.
